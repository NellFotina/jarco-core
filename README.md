# Jarco Core — Универсальное ядро для сайтов Jarco Group

![WordPress](https://img.shields.io/badge/WordPress-6.7%2B-blue?logo=wordpress)
![PHP](https://img.shields.io/badge/PHP-8.4-777BB4?logo=php)

**Jarco Core** — это высокопроизводительная родительская тема (Parent Theme), разработанная для обеспечения единой логики и функциональности всех веб-ресурсов группы компаний Jarco.

## Ключевые возможности
- **Автоматические обновления:** Интеграция с GitHub для бесшовного обновления всех сайтов сети из одного репозитория.
- **Оптимизация под WP 6.7:** Полная совместимость с новейшими стандартами ядра WordPress.
- **Универсальные опции:** Интеграция с Redux Framework для гибкого управления настройками брендов.
- **SMTP Ready:** Встроенная поддержка безопасной отправки почты.
- **SEO & Performance:** Чистый код, оптимизированная загрузка скриптов (Swiper.js) и отсутствие лишних RSS-фидов.

## Установка
1. Клонируйте репозиторий в папку `/wp-content/themes/jarco-core/`.
2. Используйте дочерние темы (Child Themes) для настройки дизайна конкретных брендов (например, Baba Shura).
3. Настройте параметры SMTP в файле `wp-config.php`.

## Структура проекта
- `/assets` — оптимизированные JS и CSS ресурсы.
- `/inc` — ядро функциональности (Redux, Переводы, Update Checker).
- `functions.php` — основной файл инициализации логики.
